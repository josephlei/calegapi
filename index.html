Open Sourcing California Legislative Information

Have you ever wondered 'how many bills were passed last year' or
'how did my senator vote on issues I care about?'

Now you can!

We've opened the silo!

Historically, CA's legislative source (leginfo.legislature.ca.gov) has provided basic search on items such as: Bill Number, House, Code Section; however this cannot be easily used for building technology apps, and does not expose granular data that already exists.

Our Solution

CA Leg API Services

Accessible

The entire CA Leg database made available over REST API as json, csv and xml

Granular

For example, no more summary counts. One row, per vote, per legislator, per bill, per revision

Updated

We are working hard to provide daily updates in the next iteration

Open

Database, API, website, this entire project made 100% with open source, no proprietary cost to taxpayers
